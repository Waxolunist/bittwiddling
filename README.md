#About

Java library for mathematics and bit twiddling, mainly inspired by [http://graphics.stanford.edu/~seander/bithacks.html](http://graphics.stanford.edu/~seander/bithacks.html).

#Examples 

For examples please see the tests or this projects homepage: [https://github.com/Waxolunist/bittwiddling](https://github.com/Waxolunist/bittwiddling).
